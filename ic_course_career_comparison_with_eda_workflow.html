數位 IC、類比 IC 與出路比較整理

根據課程描述、課程進度與就業方向整理

補充圖解：數位 IC 與類比 IC 的差異

數位 IC

重點是邏輯閘、暫存器、FSM、CPU、Cache、RTL 與 Synthesis。

類比 IC

重點是放大器、電流鏡、OP、ADC/DAC、PLL、雜訊與頻寬。

圖解重點：數位 IC 主要處理 0/1 邏輯與時序；類比 IC 主要處理連續訊號、增益、頻寬、雜訊與偏壓。

一、CMOS VLSI Full-Custom Design 課程判斷

這門課程內容包含 Full-Custom Design、CMOS VLSI、元件、電晶體、邏輯與系統層面， 也包含模擬、佈局、寄生參數萃取以及佈局後驗證。

結論：這門課是「數位 IC 為主的 Full-Custom CMOS VLSI 設計課」，不是純類比 IC。
判斷依據 說明 偏向
Full-Custom Design 數位與類比都可能使用 full-custom，但本課後續主題偏 CMOS 邏輯設計。 數位 / 類比皆可能
CMOS VLSI 通常是數位積體電路設計的核心課程。 數位 IC
Logic、System 層面 明顯與數位邏輯、數位系統設計相關。 數位 IC
Speed、Power 速度與功耗是數位 VLSI 設計的重要最佳化目標。 數位 IC
Sequencing 通常指 latch、flip-flop、register、clocking 等時序電路。 數位 IC
Circuit Families 常見內容包含 Static CMOS、Dynamic CMOS、Domino Logic、Pass Transistor Logic。 數位 IC
Memories 通常包含 SRAM、DRAM、ROM、Memory Cell 等。 數位 IC

二、VLSI System Design 課程判斷

這門課明確提到 Verilog、Combinational Logic、Sequential Logic、Non-Pipelined CPU、 Pipelined CPU、Cache、Synthesis、Simulation、Verification、DFT。

結論：VLSI System Design 是 100% 數位 IC / 數位 VLSI 系統設計。

三、兩門課的差異

課程 主要方向 學習重點
CMOS VLSI Full-Custom Design 數位 IC，但偏底層電晶體與 layout CMOS、layout、delay、power、circuit families、memory
VLSI System Design 數位 IC，偏 RTL / 系統設計 Verilog、CPU、pipeline、cache、synthesis、verification
簡單說：前者比較像「用電晶體做數位電路」，後者比較像「用 Verilog 做數位系統」。

五、如果目標是比較好找工作

首選：數位 IC 設計 / 數位 IC 驗證

原因：

  • 職缺最多
  • 和 Verilog、FPGA、CPU、VLSI System Design 關聯最大
  • 容易銜接碩士研究與業界職缺
  • AI chip、NPU、ASIC、SoC 需求仍然高

相關技能：

Verilog SystemVerilog RTL Design FPGA ASIC Computer Architecture Synthesis Verification
如果以「就業穩定、職缺多、發展性好」來看，數位 IC 是最推薦的方向。

六、如果目標是最高薪

首選：類比 IC / RF IC

優點：

  • 人才稀少
  • 技術門檻高
  • 高階工程師薪資潛力很高
  • 不容易被自動化完全取代

缺點：

  • 入門難度最高
  • 職缺數量比數位 IC 少
  • 需要很強的電路直覺與半導體元件觀念

補充：類比 IC 課程範例

類比 IC 課程通常和前面的 Digital IC / VLSI System Design 不同，重點不是 Verilog、CPU 或 Synthesis， 而是從 MOSFET 小訊號模型、偏壓電路、放大器、電流鏡、差動對、頻率響應、回授與 OP Amp 開始， 最後進入 ADC、DAC、PLL、LDO、Bandgap Reference 等類比積體電路設計。

類比 IC 圖解

處理連續訊號，重點是增益、頻寬、雜訊、線性度、穩定度與功耗。

連續訊號

類比 IC 不是只看 0/1，而是分析波形大小、頻率、相位、雜訊與失真。

類比 IC 典型課程內容

主題 學習內容 常見電路
MOSFET 元件模型 大訊號模型、小訊號模型、gm、ro、body effect NMOS / PMOS
偏壓電路 建立穩定工作點，控制電流與電壓 Current Mirror、Bias Network
單級放大器 分析增益、輸入阻抗、輸出阻抗、頻寬 Common Source、Common Gate、Source Follower

運算放大器 兩級 OP、補償、增益、頻寬、slew rate OP Amp
資料轉換器與特殊電路 類比與數位訊號轉換、時脈與電源管理 ADC、DAC、PLL、LDO、Bandgap

補充：EDA 在數位 IC 與類比 IC 的工作技術流程圖

EDA 是 Electronic Design Automation，也就是「電子設計自動化工具」。 它本身不是單純數位 IC，也不是單純類比 IC，而是支援 IC 設計流程的軟體工具與自動化技術。 數位 IC、類比 IC、Full-Custom IC、SoC、Memory 設計都會使用 EDA。

EDA 技術在數位 IC 與類比 IC 的詳細比較

EDA 技術 在數位 IC 的用途 在類比 IC 的用途 差異重點
Design Entry 設計輸入 主要輸入 Verilog、SystemVerilog、RTL code、constraint file。 主要輸入 schematic、transistor-level circuit、device size。 數位偏程式描述硬體；類比偏電晶體電路圖與元件尺寸。
Simulation 模擬 做 RTL simulation、gate-level simulation，確認邏輯功能正確。 做 DC、AC、Transient、Noise、Corner、Monte Carlo 模擬。 數位看 0/1 功能與時序；類比看波形、增益、頻寬、雜訊、穩定度。
Synthesis 邏輯合成 把 RTL 轉成 gate-level netlist，是數位 IC 核心流程。 一般類比 IC 不做 RTL synthesis。 Synthesis 幾乎是數位 IC 專用流程。
STA 靜態時序分析 檢查 setup time、hold time、clock path、timing closure。 類比 IC 通常不以 STA 為主要分析方式。 STA 是數位 IC 判斷速度是否達標的核心工具。
Place & Route 自動布局繞線 自動放置 standard cell，進行 clock tree synthesis 與 routing。 類比 layout 多半需要人工精修，強調 matching、symmetry、guard ring。 數位可高度自動化；類比較依賴工程師手動 layout 經驗。
DRC 設計規則檢查 檢查數位 layout 是否違反製程規則。 檢查類比 layout 是否違反製程規則。 兩者都需要，是 tape-out 前必要檢查。
LVS 版圖電路比對 確認 layout 產生的 netlist 與原本電路或 gate netlist 一致。 確認 layout 與 schematic 的 transistor connection 一致。 兩者都需要，但類比 LVS 常更重視 transistor-level connection。
PEX 寄生參數萃取 萃取 RC parasitic，用於 timing、power 與 signoff analysis。 萃取寄生電阻、電容，回到 Spectre 做 post-layout simulation。 數位偏時序收斂；類比偏波形、增益、相位裕度是否被寄生影響。
Power Analysis 功耗分析 分析 dynamic power、leakage power、clock power。 分析 bias current、static power、signal swing、效率與熱效應。 數位功耗和切換率、頻率、負載電容有關；類比功耗常和偏壓電流直接相關。
Formal Verification 形式驗證 用數學方法確認 RTL 與 netlist 邏輯等價。 類比 IC 較少使用傳統 formal equivalence。 Formal verification 主要偏數位 IC。
DFT / ATPG 測試設計 加入 scan chain、BIST、ATPG pattern，提升晶片可測試性。 類比也有測試需求，但方式常是量測電壓、電流、頻率、增益等規格。 DFT 是數位量產測試的重要流程；類比測試更偏規格量測。
Signoff 簽核 確認 timing、power、IR drop、EM、DRC、LVS 都通過。 確認 post-sim、corner、noise、stability、DRC、LVS 都通過。 兩者都需要 signoff，但檢查重點不同。
總結：EDA 是 IC 設計的工具與自動化技術。若工具重點是 RTL、Synthesis、STA、P&R、DFT， 通常偏數位 IC；若工具重點是 Schematic、SPICE、Virtuoso Layout、DRC/LVS、PEX、Post-Sim， 通常偏類比 IC 或 Full-Custom IC。
